Opportunity
Standard

The Opportunity Standard is the design system for NYC Opportunity's digital products. It is ideally suited for building catalogs of City services and online applications, providing simple and flexible CSS and JavaScript-based components and utilities for building user interfaces quickly and easily.

Installation

Installing via NPM will store Opportunity Standard source and distribution files in your project's node_modules directory under @nycopportunity/standard.

Source; ES (JavaScript), Sass (stylesheet), and uncompressed SVGs will be in the src/ directory. Distribution; concantenated JavaScript, CSS stylesheets, and optimized SVGs and SVG sprites will be in the dist/ directory. Further integration details can be found in the installation guide.

Read the installation guide

Alternatively, view the quick start documentation to use the CDN method of installation.
